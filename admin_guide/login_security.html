Login & Security
Accessing the Admin Dashboard
1Navigate to Admin Portal

Access at https://admin.lexova.com using a secure browser.

2Enter Credentials

Enter your admin email and password (separate from regular accounts).

3Two-Factor Authentication

Enter the verification code sent to your mobile device or authentication app.

4Session Management

Sessions expire after 30 minutes of inactivity. Use "Keep me logged in" only on secure devices.

Account Management

To update your admin profile or change security settings:

Click your profile icon in the top-right corner

Select "Account Settings" from the dropdown

Update profile, change password, or manage 2FA

Security Best Practices
Strong Password Policy

Use a unique password with at least 12 characters, including uppercase, lowercase, numbers, and special characters.

Secure Environment

Only access from trusted devices and networks. Avoid public Wi-Fi for administrative tasks.

Regular Password Updates

Change your admin password every 90 days and never reuse previous passwords.

Always Log Out

Always use the "Log Out" button when finished, especially on shared computers.
